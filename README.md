# Week7-ExpenseTracker

Program Name: Expense Tracker

Purpose:
This program helps users record their daily expenses by entering a description and amount.
It stores all expenses in a list and allows users to view each expense or the total amount spent.

Features:
- Add multiple expenses.
- View list of all expenses.
- View the total expense.
- Simple menu navigation.

Technologies Used:
- Classes and objects (Expense class)
- Lists for dynamic storage
- Loops and conditional statements
- Input validation using TryParse

Sample Usage:
1. Add Expense: Grocery - $20
2. Add Expense: Transport - $15
3. View Expenses: Shows both entries
4. View Total: $35

This simulates a real-world scenario for tracking personal finances.
